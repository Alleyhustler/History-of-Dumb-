        C eh uma linguagem compilada, ou seja, para criar/executar um programa
        em c, vc cria o codigo-fonte atraves de um editor de textos comum e
        depois manda para um compilador, que vai transforma-lo em codigoobjeto
        e vai gerar o excutavel. Isso quer dizer q para executar o programa vc
        precisa de um compilador, vc pode baixar algum gratuito da net.

-tipos de dados:
char            1byte    -127  a 127
unsigned char   1byte      0   a 255
int             2bytes  -32767 a 32767
unsigned int    2bytes     0   a 65535
float           4bytes
... e mais uns outros 
* o numero de bytes pode variar de acordo com o processador

a diferenca entre char/unsigned char e int/unsigned int eh a forma como o bit
mais significativo eh tratado. se for signed, quer dizer q o bit mais
signifativo indica o sinal, caso contrario sera parte do numero.

-variaveis
para se declarar uma variavel, a sintaxe eh:
tipo_de_dado nome_da_variavel;

exemplos:
char letra;
int unidades;
float preco;
etc.

lembrando q toda sentenca em C deve terminar com ';' .

variaveis podem ser declaradas dentro de funcoes, nos parametros de funcoes,
ou fora delas, de acordo com o proposito.

variaveis locais: sao declaradas dentro de funcoes e so podem ser
referenciadas no bloco de codigo ao qual pertencem, pois sao automaticamente
destruidas.
ex:
void main() {        <-- isso aki significa q comeca uma funcao chamada main
char z;              <-- variavel de nome 'z' do tipo caracter declarada
int a,b;             <-- variaveis inteiro 'a' e 'b'
a=0;                 <-- 'a' passa a valor 0
b=1;                 <-- 'b'  "        "   1
        if(a<b) {    <-- se 'a' for menor q 'b' executar bloco de codigo
         char x;     <-- declarada no inicio de um bloco de codigo
         comando1;
         comando2;
        }            <-- ao fim do bloco a variavel local 'x' eh destruida

}                    <-- termino da funcao main, pra todos efeitos tb eh um
                         bloco de codigo, e a variavel 'z' eh destruida.

variaveis locais como parametros: sao declaradas depois do nome da funcao,
entre os parenteses; apresentam o mesmo comportamento das variaveis locais,
mas sao iniciadas com o valor do parametro passado a funcao.
ex:
         int soma(int a, int b) {    /* variaveis criadas */
          retur a*b;                    
         }
         main() {
          soma(5,4);                 /* chama a funcao soma, que vai atribuir
         }                            5 a variavel 'a' e 4 a variavel 'b'.
                                     */
        

variaveis globais: essas variaveis globais sao declaradas fora de qualquer
funcao e podem ser acessadas (lidas/modificadas) de qualquer parte do codigo.
elas sao uteis quando o mesmo dado eh usado em diversas partes (blocos ou
funcoes) do programa.
ex:
        int a,b;              /* eh a mesma funcao do exemplo acima
        int soma(void) {       so q usando variaveis globais. note
         return a*b;           q do jeito q esta a funcao nao eh muito
        }                      eficiente, as variaveis 'a' e 'b' vao ficar
                               ocupando memoria e so sao usadas na funcao
                               soma(), mas o codigo eh valido.
                               */
ex1:
   int municao;
   void planejamento() {
    ... operacoes envolvendo o valor de municao
   }
   void ataque() {
   ... funcao q usa o valor de municao
   }
   void mostar_status() {
   ... precisa de 'municao'
   }
   etc                          /* desculpem o exemplo escroto ;) */

        if(a<b) {

-funcoes
 as funcoes podem receber eou retornar valores, e seu prototipo deve ser 
 declarado antes da funcao main().
 forma:
 tipo_valor_retorno funcao( variavel );

 tipo_valor_retorno eh o tipo de dado que ele retorna(char,int,etc).
 variavel deve ser a declaracao da variavel-parametro. quando a funcao
 eh chamada, eh atribuida a variavel o valor do parametro de chamada.

ex:
        int soma(int a,int b);  
        void main(void) {
         int resultado; 
          resultado = soma(5,4); 
                /* aproveitando isso aqui.. em C, pode-se fazer coisas do tipo
                   resultado = soma(soma(5,4),2); ou
                   resultado = soma(soma(1,2),soma(5,0));
                
                   ou entao , por exemplo, se resultado soh fosse usado 
                   para imprimir o resultado, poderia-se jogar direto
                   o valor que a funcao soma() retorna como parametro
                   de outra funcao, 
                   tipo: printf("%d",soma(4,5));
                */
        }
        int soma(int a,int b) {
        return a+b;
        }
